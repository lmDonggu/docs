# YAML front matter 页面配置

# title

  • description: 文章标题，放弃通过一级目录定义标题的方式，改在 Front Matter 中定义。
  title: 文件配置模板
1

# description

  • 当前页面的描述。
description: "模板文件"
1

# lang

  • 当前页面的语言
lang = "en-US"
1

# date

  • description: 文章创建日期，格式 2019-08-08 或 2019-08-08 08:08:08。
date: 2021-05-13
1

# sidebar

  • description: 是否开启侧边栏。
sidebar: 'auto'
1

# subSidebar

  • YAML front matter 来禁用指定页面的侧边栏
subSidebar: false
1

# categories

  • description: 所属分类。
categories:
 - 烹饪
 - 爱好
1
2
3

# tags

  • description: 所属标签。
tags:
 - 烤
 - 鸭子
1
2
3

# author

  • 为单篇文章设置作者姓名
---
title: 你还没真的努力过，就轻易输给了懒惰
date: 2015-04-23
categories: article
author: 渡渡

# Vuepress Plugin Auto Sidebar (opens new window)插件文件的配置

# 上个文件链接

autoPrev: other-filename
1

# 下个文件链接

autoNext: other-filename
1

# 提取二级标题

sidebarDepth: 2
1

# 不显示侧边栏

autoIgnore: false
1

# 分组方法其中 - 和 + 二者选其一，- 代表在默认分组的下方，+ 代表在默认分组的上方，符号后面的数字会进一步决定分组的排序。

autoGroup-2: 数组方法 
// 或
autoGroup+10: group10
